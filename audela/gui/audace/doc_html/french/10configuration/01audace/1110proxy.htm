Internet (Proxy)

Les paramètres du serveur Proxy sont stockés dans 4 variables globales qui sont elles-mêmes sauvegardées dans le fichier de configuration "audace.ini".

Ces 4 variables globales sont :
- Pour le nom du Proxy : conf(proxy,host)
- Pour le port du Proxy : conf(proxy,port)
- Pour l'utilisateur du Proxy : conf(proxy,user)
- Pour  le mot de passe du Proxy : conf(proxy,password)
