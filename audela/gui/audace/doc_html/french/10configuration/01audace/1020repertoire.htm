2. Fonction des boutons

Pour faciliter la tâche, le libellé du champ sélectionné apparaît en italique et encadré.

Le plus simple est de naviguer dans la fenêtre et de désigner le bon dossier avec la souris puis d'appuyer sur le bouton OK.

3. Configuration du répertoire images

Après configuration du chemin de ce dossier, des options sont possibles :

  • Pas de sous-répertoire : Toutes les images créées se trouveront directement dans ce dossier.
  • Nom du sous-répertoire : Toutes les images créées se trouveront dans le sous-répertoire, par défaut son nom est la date du jour.
  • Date du jour : Un répertoire est créé chaque jour dans le dossier images, ce répertoire change de nom automatiquement soit Ã  0 h, soit à 12 h selon le choix fait.

5. Configuration des répertoires catalogues

Au premier lancement d'Aud'ACE le chemin de chaque catalogue désigne un dossier dans xxxx\Application Data\Audela\catalog.

Il est possible de modifier cette localisation en sélectionnant Catalogues dans la fenêtre de sélection Ã  gauche et en désignant le nouveau dossier à l'aide du bouton [...] à droite. Le nouveau chemin s'affiche au milieu. Tous les catalogues peuvent être installés dans ce dossier mais ce n'est pas une obligation. Il est possible de définir un chemin différent pour chaque catalogue en sélectionnant son nom symbolique, à l'exemple de MICROCAT.

Si plusieurs catalogues sont installés dans le même dossier, la sélection du nom symbolique Base et la désignation de sa localisation lance la recherche automatique des catalogues présents dans ce dossier. Cette commande permet d'explorer plusieurs répertoires.

Le nom des dossiers des catalogues peut être en majuscules ou en minuscules et comporter des "_", mais pas des "-" ou des "."

6. Ajout/Suppression d'un catalogue personnel

La liste des catalogues définie par défaut ne peut être modifiée.
Il est toujours possible de ne pas configurer les chemins d'accès si les catalogues ne sont pas utilisés.
Pour ajouter un catalogue ne figurant pas dans la liste (ex.: AAVSO, MPC, BOWELL), il suffit de :

  • sélectionner un nom de catalogue quelconque dans la liste, à l'exception de Base et de catalogues,
  • de le remplacer par le nom souhaité (seuls les caractères alphanumériques et les underscores sont admis),
  • puis de cliquer deux fois sur le sélecteur de la boîte. Le nom choisi s'affiche en majuscules en fin de liste,
  • configurer le chemin à l'aide du bourton ....
  • cliquer sur Appliquer.
Pour supprimer un catalogue personnel, il suffit de supprimer le nom du catalogue et de cliquer deux fois sur le sélecteur.
La liste est mise à jour. Le répertoire et son contenu peuvent être effacés manuellement.

7. Configuration manuelle du chemin des répertoires

Si vous êtes un fanatique des commandes en ligne, vous pouvez éditer directement le chemin des répertoires au clavier dans les champs, mais faites très attention les \ habituels de Windows doivent impérativement être remplacés par des / dans la boîte de dialogue Répertoires.
