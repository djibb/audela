1Présentation

Ce guide s'adresse aux développeurs de programmes qui souhaitent utiliser une Astrobrick dans leur programme.

Une Astrobrick est une librairie dynamique écrite en C++ pour Audela pouvant être utilisée par d'autres programmes écrits dans différents langages (C++, Tcl, Python, C#) et pouvant être exécutée sous Window, Linux et Macosx.


Chaque ASTROBRICK est fournie avec:

  • un fichier binaire pour chaque système d'exploitation Windows, Linux et Macosx ( .dll , .so , .dylib)

  • un fichier header .h et un fichier .lib pour l'utilisation par un programme écrit en C++

  • un exemple de fichier .cs pour l'utilisation par un programme écrit en C#

  • un exemple de fichier .py pour l'utilisation par un programme écrit en Python 3

  • une librairie complémentaire pour l'utilisation par un programme écrit en Tcl 8.6 (cette librairie complémentaire sert à convertir tous les paramètres des fonctions en chaînes de caractères, seul format accepté pat l'interpréteur Tcl)


2Exemple libsimple

L'astrobrick libsimple contient les fonctions suivantes: 

int Ilibsimple_processAdd(int a, int b); 
int Ilibsimple_processSub(int a, int b);


Cette Artrobrick est fournie avec les fichiers:

libsimple.dll

libsimple.so

libsimple.dylib

corps de l'astrobrick pour Windows, Linux et Macosx

libsimple.h

libsimple.lib

déclaration des fonctions pour un programme C++

libsimple.cs

exemple d'utilisation pour un programme C#

libsimple.py

exemple d'utilisation pour un programme Python

libsimple_tcl.dll

libsimple_tcl.so

libsimple_tcl.dylib

pkgIndex.tcl

librairie TCL complémentaire et script de chargement pkgIndex.tcl pour utilisation dans un programme Tcl

libsimple.html

Documentation




3Utilisation dans un programme C++

Exemple de source C++ utilisant libsimple

// libsimple_test.cpp 

#include "libsimple.h"

int main(int argc, char* argv[])
{
   int a = 8;
   int b = 4;

   int result = Ilibsimple_processAdd(a, b);
   cout << "Ilibsimple_processAdd: " << a << " + " << b << " = " << result << endl;
   // printf(“"Ilibsimple_processAdd: %d + %d = %d\n”,a, b, result);

   int result2 = Ilibsimple_processSub(a, b);
   cout << "processAdd: " << a << " - " << b << " = " << result2 << endl;
   // printf(“"Ilibsimple_processSub: %d + %d = %d\n”,a, b, result2);
}


Le programme C++ importe le fichier libsimple.h pour pourvoir utiliser les fonctions de libsimple


      3.1 Windows Visual C++

Compilation du projet libsimple_test.vcproj

  • ajouter le répertoire de libsimple.h dans les paramètres du compilateur

  • ajouter libsimple.lib dans les paramètres du linker


Résultat

>libsimple_test.exe
Ilibsimple_processAdd: 8 + 4 = 12
Ilibsimple_processSub: 8 - 4 = 4

>


3.2 Linux

Compilation :

  • ajouter le répertoire de libsimple.h dans les paramètres de compilation du programme ( paramètre -I )

  • ajouter libsimple.so dans les paramètres du linker (paramètres -L et -l )

$ make 
*** Compiling libsimple_test.cpp
g++ -O2  -fPIC  -fno-stack-protector -c -Wall  -I../../libsimple/src -o libsimple_test.o ../src/libsimple_test.cpp
*** Linking library libsimple_test
gcc libsimple_test.o -ldl -lm -lstdc++ -Wl,-rpath,. -L../bin  -lsimple -o libsimple_test 
$


Résultat

$ ../bin/libsimple_test 
Ilibsimple_processAdd: 8 + 4 = 12
Ilibsimple_processSub: 8 - 4 = 4

$



      3.3 Macosx

Compilation :

  • ajouter le répertoire de libsimple.h dans les paramètres du compilateur ( paramètre -I )

  • ajouter libsimple.dylib dans les paramètres du linker (paramètres -L et -l )


$ make

Résultat

Ilibsimple_processSub: 8 - 4 = 4

$



4Utilisation dans un programme C#

Exemple de source utilisant libsimple (projet audela-csharp.csproj )


// audela_csharp.cs

using System;
using System.Text;

namespace audela_csharp
{
    class libsimple_test
    {
        static void Main(string[] args)
        {
            int a = 4;
            int b = 8;

            int result = Ilibsimple.Ilibsimple_processAdd(a, b);
            Console.WriteLine("Ilibsimple_processAdd: " + a + " + " + b + " = " + result);

            int result2 = Ilibsimple.Ilibsimple_processSub(a, b);
            Console.WriteLine("Ilibsimple_processSub: " + a + " + " + b + " = " + result2);

        }
    }
}


Le programme C# doit être compilé avec le fichier libsimple.cs pour pourvoir utiliser les fonctions de libsimple

4.1 Windows Visual C#

Compilation du projet audela-csharp.csproj

  • ajouter libsimple.cs dans le projet


Exécution

  • copier libsimple.dll dans le même répertoire que audela-csharp.exe

  • lancer audela-csharp.exe


Résultat

> audela_csharp.exe
Ilibsimple_processAdd: 8 + 4 = 12
Ilibsimple_processSub: 8 - 4 = 4

>


4.2 Linux Mono C#

Compilation :

  • compiler le programme audela-csharp.cs avec libsimple.cs 


$ cd audela-csharp/linux 
$ make


Exécution

  • copier libsimple.so dans le même répertoire que audela-csharp

  • lancer audela-csharp


Résultat

$ ./audela_csharp 
Ilibsimple_processAdd: 8 + 4 = 12
Ilibsimple_processSub: 8 - 4 = 4

$



5Utilisation dans un programme Python

Exemple de source Python utilisant libsimple

# file: audela.py

import libsimple 

a = 8
b = 4
result = libsimple.processAdd(a,b)
print ("ILibsimple.processAdd : %d + %d = %d" % (a,b,result))

result2 = libsimple.processSub(a,b)
print ("ILibsimple.processAdd : %d - %d = %d" % (a,b,result2))


Le programme python audela.py importe le fichier libsimple.py qui montre comment charger la librairie


5.1 Windows

Exécution

  • copier libsimple.py dans le répertoire audela-python

  • copier libsimple.dll dans le répertoire audela-python\libs

  • lancer l'exécution de audela.py


Résultat

> python.exe audela.py
Ilibsimple_processAdd: 8 + 4 = 12
Ilibsimple_processSub: 8 - 4 = 4

>>>
>


5.2 Linux

Exécution

  • copier libsimple.py dans le répertoire audela-python

  • copier libsimple.so dans le répertoire audela-python\libs

  • lancer l'exécution de audela.py


Résultat

$ python3 audela.py 
Ilibsimple_processAdd: 8 + 4 = 12
Ilibsimple_processSub: 8 - 4 = 4

>>>
$



6Utilisation dans un programme Tcl


Exemple de source TCL utilisant libsimple :


#--- file: libsimple_test.tcl

package require libsimple_tcl

set a 8
set b 4
set result [ Ilibsimple_processAdd $a $b ]
console::disp "Ilibsimple_processAdd  : $a + $b = $result \n"

set result2 [ Ilibsimple_processSub $a $b ]
console::disp "Ilibsimple_processAdd : $a + $b = $result2 \n"


Le programme libsimple_test.tcl importe le package libsimple_tcl qui charge la librairie.

5.1 Windows

Exécution avec Audela


  • créer le répertoire audela\lib\libsimple

  • copier dans ce répertoire libsimple_tcl.dll , libsimple.dll et pkgIndex.tcl

  • copier libsimple.tcl dans le répertoire de travail de Audela

  • lancer l'exécution de Audela

  • lancer “source libsimple.tcl” dans la console de Audela


Résultat

> source libsimple.tcl
Ilibsimple_processAdd: 8 + 4 = 12
Ilibsimple_processSub: 8 - 4 = 4

>


5.2 Linux

Exécution avec Audela


  • créer le répertoire audela/lib/libsimple

  • copier dans ce répertoire libsimple_tcl.so libsimple.so et pkgIndex.tcl

  • copier libsimple.tcl dans le répertoire de travail de Audela

  • lancer l'exécution de Audela

  • lancer “source libsimple.tcl” dans la console de Audela


Résultat

> source libsimple.tcl
Ilibsimple_processAdd: 8 + 4 = 12
Ilibsimple_processSub: 8 - 4 = 4

>



Exécution avec l'interpréteur Tcl standard de Linux

  • créer le répertoire tcl86/lib/libsimple

  • copier dans ce répertoire libsimple_tcl.so , libsimple.so et pkgIndex.tcl

  • copier libsimple.tcl dans un répertoire de travail

  • remplacer “console::disp” par “puts” dans libsimple.tcl

  • lancer l'exécution de tclsh depuis ce répertoire de travail

  • lancer “source libsimple.tcl” dans la console de Tcl


Résultat

$ tclsh
% source libsimple.tcl
Ilibsimple_processAdd: 8 + 4 = 12
Ilibsimple_processSub: 8 - 4 = 4

% exit
$
