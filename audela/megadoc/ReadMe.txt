Le répertoire "megadoc" contient les fichiers de configuration de Doxygen pour créer la documentation de AudeLA.
Toute modification intempestive peut créer des effets indésirables.

Le contenu du répertoire "megadoc" :

- Readme.txt (ce fichier),
- audace.doxyfile contient la configuration de Doxygen,
- copyimage.bat copie les images des aides des répertoires "../french/images" et "doxy_conf/images"
  noter que copyimae.bat recopie aussi les sous-répertoires de images.

- mainpage.txt contient le sommaire de la page de navigation principale
- 1_page.txt contient le détail des rubriques Premiers pas, Fichier et Affichage
- 2_page.txt contient le détail de la rubrique Images
- 3_page.txt contient le détail des rubriques Analyse, Caméra et Télescope
- 4_page.txt contient le détail de la rubrique Configuration
- 5_page.txt contient le détail de la rubrique Aide
- 6_page.txt contient le détail de la navigation dans les pages récupérées dans docuwiki

- doxy_warning.log est produit par un run de Doxygen et contient des warning
  non transformés en @toto

- Doxygen en pratique.doc donnent des conseils pratiques pour bien démarrer.

- le répertoire audela_site contient les fichiers et images récupéres dans docuwiki

- le répertoire doxy_conf contient le nécessaire (header, css, image) pour configurer les pages doxygen comme celles du site.
