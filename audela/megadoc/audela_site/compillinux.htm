Voilà comment compiler AudeLA sous son OS préféré :

Prérequis :

  • xorg-dev
  • gcc et g++
  • libstdc++
  • tcl-dev
  • tk-dev
  • libgsl0-dev
  • linux-headers
  • libusb-dev

L'installation des paquets *-dev engendrera l'installation des librairies binaires. Par exemple sous Debian, en tant que root que pour cette étape : 

 sudo apt-get install tcl8.6-dev tk8.6-dev gcc g++ make libgsl0-dev blt subversion xorg-devlinux-libc-dev libstdc++6-4.7-dev libtiff5-dev libftdi-dev libv4l-dev

Pour télécharger AudeLA pour Linux, il y a 2 méthodes :

  • soit télécharger l'archive de la version stable.
  • soit télécharger la mise à jour depuis le serveur SVN (dernière version en développement, donc potentiellement instable mais possédant les nouveautés) avec l'identifiant de son compte :

Première mise à jour : 

  mkdir audela-svn ; cd audela-svn
  svn co -q https://audela.svn.sourceforge.net/svnroot/audela/trunk/audela audela

Mises à jour suivantes : 

 svn co -q https://audela.svn.sourceforge.net/svnroot/audela/trunk/audela audela

Il peut s'avérer nécessaire au préalable de rendre exécutable ce fichier : 

 cd audela/src
 chmod u+x configure
 ./configure
 ou
 ./configure --with-tcl=/usr/share/tcltk/tcl8.5 --with-tk=/usr/share/tcltk/tk8.5

 make external && make contrib && make

Rendre exécutable le fichier audela.sh : 

 cd ../bin
 chmod u+x audela.sh

Lancer AudeLA : 

 ./audela.sh

Et c'est parti
