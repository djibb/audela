Le pilotage d'un télescope constitue un atout très important en imagerie CCD. Il est ainsi possible d'acquérir les images sans perdre de temps avec la recherche du champ à observer. Evidemment, pour pouvoir piloter un télescope avec Audace, il faut qu'il soit motorisé et équipé d'une interface reconnue par Audela. Si le protocole de communication de votre télescope n'est pas implanté dans Audela, vous pouvez le faire vous même. Pour cela, voir la page consacrée à la réalisation d'un script de driver de télescope.

1.Paramétrer le télescope

Pour régler les paramètres de connexion d'un télescope, il faut faire apparaître la fenêtre des réglages. Pour cela, aller dans le menu Réglages puis cliquer sur Télescope :

La fenêtre de configuration du télescope se présente comme ci dessous.

Après avoir rempli les champs avec les valeurs correspondant à votre télescope, appuyer sur le bouton OK.

2. Pointer le télescope

Pour faire pointer le télescope, il faut faire apparaître le panneau télescope. Pour cela, aller dans le menu Panneau puis cliquer sur Telescope :

Le panneau de télescope sert à régler les paramètres de pointage et de lancer une pose de vérification.

Le pointage est paramétré grâce a un menu déroulant. Par défaut l'option "Coord." permet d'entrer manuellement la valeur coordonnées (ascension droite et déclinaison, appelées par la suite RA et DEC). Le menu propose aussi une liste d'étoiles brillantes utiles pour initialiser le télescope. Enfin, il sera bientôt possible de choisir un objet par son numéro Messier par exemple.

Le cartouche des coordonnées montre la valeur de RA et DEC pointées par le télescope. Le rafraichissement de ces coordonnées n'est pas automatique (il est réalisé à chaque déplacement du télescope effectué avec ce panneau de contrôle). Pour rafraichir, cliquer sur les coordonnées. Si la connexion entre Audace et le télescope n'a pas été établie, les coordonnées n'apparaissent pas. Il est cependant possible de connecter directement le télescope en cliquant dans cette zone.

La raquette permet un déplacement dans les quatre directions. La vitesse de déplacement est indiquée par un chiffre situé au milieu des boutons. Plus ce chiffre est grand, plus la vitesse est importante. On change de vitesse en appuyant sur le chiffre.

Le bouton MATCH sert à re-initialiser les coordonnées du télescope sur les coordonnées indiquées dans la zone d'entrée orange du cartouche de pointage.

Le cartouche du bas sert à paramétrer et lancer une pose CCD pour vérifier le centrage de l'objet. Il faut remplir le temps de pose et la définition de l'image. La haute définition consiste à convertir un pixel du composant CCD en un pixel dans l'image finale (encore appelé binning 1x1). La moyenne définition consite à rassembler les pixels du composant CCD par paquets de quatre dans l'image finale (encore appelé binning 2x2). La basse définition consiste à rassembler les pixels du composant CCD par paquets de seize dans l'image finale (encore appelé binning 4x4). Plus la définition est haute, meilleure sera la résolution, mais l'image occupera une plus grande place mémoire et sera plus longue à être lue. Pour la phase de vérification de pointage on utilise généralement la basse définition. Le bouton GO permet de démarrer l'acquisition d'image. A la fin de la lecture, l'image est automatiquement affichée.
