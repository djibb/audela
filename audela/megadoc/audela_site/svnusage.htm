Aide-mémoire des commandes Subversion.

Sous GNU/Linux

Téléchargement des sources d'Audela :

  • Installer les paquets esvn et subversion,
  • Créer un r√©pertoire audela-svn et entrer dedans,
  • Exécuter la commande de téléchargement total de la version svn d'audela : 
    svn co https://audela.svn.sourceforge.net/svnroot/audela/trunk/audela audela
  • Rentrer son identifiant ainsi que son mot de passe

Mise à jour locales et commit des modifications :

  • Ajouter la déclaration des types dans le fichier .subversion/config (qui a été créé lors de la première exécution de la commande svn) :
    •    enable-auto-props = yes
   [auto-props]
   *.c = svn:eol-style=native
   *.cpp = svn:eol-style=native
   *.h = svn:keywords=Author Date Id Rev URL;svn:eol-style=native
   *.dsp = svn:eol-style=CRLF
   *.dsw = svn:eol-style=CRLF
   *.sh = svn:eol-style=native;svn:executable
   *.txt = svn:eol-style=native;svn:keywords=Author Date Id Rev URL;
   *.png = svn:mime-type=image/png
   *.jpg = svn:mime-type=image/jpeg
   Makefile = svn:eol-style=native
   *.tcl = svn:eol-style=native;svn:keywords=Author Date Id Rev URL;
   *.cap = svn:eol-style=native;svn:keywords=Author Date Id Rev URL;
  • Lancer esvn
  • Déclarer le répertoire audela-svn/audela comme répertoire de travail : menu Fichier/Ouvrir le répertoire de travail,
  • En cliquant droit sur un fichier ou un répertoire, on a accès aux fonctions usuelles de Commit, Update, Diff, ...
  • Si vous avez un compte SourceForge, paramètrez votre coordonnés d'authentification dans : menu Fichier/Options onglet Other puis champs User et Password.

    Sous Windows

    • Installer la bonne version du logiciel TortoiseSVN,
    • Rebooter l'ordinateur pour terminer l'installation de TortoiseSVN,
    • Créeer un répertoire vide , par exemple D:\audela-svn,
    • Depuis l'explorateur de fichier de Windows, cliquer avec le bouton DROIT de la souris sur le nom du nouveau répertoire pour faire apparaître le menu contextuel dans lequel se trouve le menu SVN Chekout,
    • Avec le menu SVN Checkout récupérer les sources de Audela à partir de l'URL https://svn.code.sf.net/p/audela/code/trunk/audela,

    Ouvrir la fenêtre de configuration de TortoiseSVN avec le menu TortoiseSVN>Settings

    Cliquer sur le bouton Edit

    modifier la configuration en mettant les mêmes paramètres que pour Linux (voir ci-dessus)
